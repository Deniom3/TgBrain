# Embeddings
Языки: [English](embeddings.md) | [Русский](embeddings_ru.md)

## Обзор

Система эмбеддингов преобразует текст в векторные представления для семантического поиска. Несколько провайдеров эмбеддингов поддерживаются через единый уровень абстракции.

## Поддерживаемые провайдеры

| Провайдер | Описание |
|----------|-------------|
| Ollama | Локальные модели эмбеддингов через сервер Ollama |
| Gemini | Модели эмбеддингов Google через API |
| OpenRouter | Модели эмбеддингов через OpenRouter |
| LM Studio | Локальные модели через сервер LM Studio |

## Размерность вектора

Размерность вектора автоматически определяется из модели эмбеддинга при первом использовании и сохраняется в базе данных. Это обеспечивает согласованность между перезапусками и позволяет отслеживать миграции.

## Управление провайдерами эмбеддингов через API

### Список всех провайдеров

```bash
curl http://localhost:8000/api/v1/settings/embedding
```

### Получение конкретного провайдера

```bash
curl http://localhost:8000/api/v1/settings/embedding/ollama
```

### Обновление настроек провайдера

```bash
curl -X PUT http://localhost:8000/api/v1/settings/embedding/ollama \
  -H "Content-Type: application/json" \
  -d '{
    "url": "http://localhost:11434",
    "model": "nomic-embed-text"
  }'
```

### Активация провайдера

```bash
curl -X POST http://localhost:8000/api/v1/settings/embedding/ollama/activate
```

### Проверка работоспособности провайдера

```bash
curl -X POST http://localhost:8000/api/v1/settings/embedding/ollama/check
```

### Обновление размерности (Ollama)

```bash
curl -X POST http://localhost:8000/api/v1/settings/embedding/ollama/refresh-dimension
```

## Эмбеддинг во время ingestion

Каждое сообщение, полученное из Telegram, автоматически проходит эмбеддинг перед сохранением. Если эмбеддинг не удаётся, сообщение помещается в таблицу `pending_messages` для последующей обработки.

## Эмбеддинг во время поиска

Когда пользователь отправляет RAG-запрос, текст запроса проходит эмбеддинг и используется для поиска похожих сообщений через HNSW-индекс pgvector.

## Реиндексация

При изменении модели или провайдера эмбеддингов все существующие эмбеддинги сообщений должны быть перегенерированы. Используйте Reindex API для запуска этого процесса. Подробности см. в [Reindex](reindex_ru.md).
