## Уровень 4: Код

Кодовая база имеет многоуровневую структуру:

```
src/
  api/              -- HTTP API роутеры и модели
  settings_api/     -- API роутеры для настроек
  application/      -- Сервисы приложения
  domain/           -- Доменные модели и объекты-значения
  infrastructure/   -- Адаптеры базы данных и внешних сервисов
  ingestion/        -- Приём сообщений из Telegram
  embeddings/       -- Реализации провайдеров эмбеддингов
  providers/        -- Реализации провайдеров LLM
  rag/              -- RAG-поиск и суммаризация
  reindex/          -- Переиндексация векторов
  rate_limiter/     -- Адаптивное ограничение частоты запросов
  schedule/         -- Планировщик по расписанию (cron-подобный)
  settings/         -- Репозитории настроек
  webhook/          -- Доставка вебхуков
  batch_import/     -- Пакетная обработка импорта
  auth/             -- QR-аутентификация
  config/           -- Управление конфигурацией
```

Зависимости импортов контролируются через import-linter для поддержания архитектурных границ.
