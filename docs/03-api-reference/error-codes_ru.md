# Коды ошибок

Языки: [English](error-codes.md) | [Русский](error-codes_ru.md)

В этом документе перечислены все коды ошибок API, сгруппированные по префиксам.

## APP-* — Ошибки приложения

| Код | HTTP Status | Описание |
|------|-------------|----------|
| APP-001 | 400 | Неверный часовой пояс |
| APP-002 | 404 | Настройка не найдена |
| APP-003 | 500 | Не удалось сохранить настройку часового пояса |

## WHK-* — Ошибки webhook

| Код | HTTP Status | Описание |
|------|-------------|----------|
| WHK-001 | 404 | Чат не найден |
| WHK-002 | 400 | Неверная конфигурация webhook |
| WHK-006 | 400/500 | Webhook не настроен или доставка не удалась |
