# LLM-провайдеры
Языки: [English](llm-providers.md) | [Русский](llm-providers_ru.md)

## Обзор

TgBrain поддерживает четыре LLM-провайдера для генерации сводок и генерации ответов RAG.

## Gemini

Модели Google Gemini через официальный API.

**Конфигурация:**
- `GEMINI_API_KEY` -- Обязательный ключ API
- `GEMINI_BASE_URL` -- Базовый URL API (по умолчанию: generativelanguage.googleapis.com)
- `GEMINI_MODEL` -- Имя модели (по умолчанию: gemini-2.5-flash)

**Случай использования:** Провайдер по умолчанию, хороший баланс качества и скорости.

## OpenRouter

Доступ к 300+ моделям через платформу OpenRouter.

**Конфигурация:**
- `OPENROUTER_API_KEY` -- Обязательный ключ API
- `OPENROUTER_BASE_URL` -- Базовый URL API (по умолчанию: openrouter.ai/api/v1)
- `OPENROUTER_MODEL` -- Имя модели (по умолчанию: auto)

**Случай использования:** Доступ к широкому разнообразию моделей, резервный провайдер.

## Ollama

Локальные модели, работающие через сервер Ollama.

**Конфигурация:**
- `OLLAMA_LLM_ENABLED` -- Включить/отключить (по умолчанию: true)
- `OLLAMA_LLM_BASE_URL` -- URL сервера Ollama (по умолчанию: http://localhost:11434)
- `OLLAMA_LLM_MODEL` -- Имя модели (по умолчанию: deepseek-coder:6.7b)

**Случай использования:** Локальный инференс, отсутствие затрат на API, полная приватность данных.

**Примечание для Docker:** Используйте `http://host.docker.internal:11434` вместо `localhost` при запуске в Docker.

## LM Studio

Локальные модели, работающие через сервер LM Studio.

**Конфигурация:**
- `LM_STUDIO_ENABLED` -- Включить/отключить (по умолчанию: false)
- `LM_STUDIO_BASE_URL` -- URL сервера (по умолчанию: http://localhost:1234/v1)
- `LM_STUDIO_MODEL` -- Имя модели
- `LM_STUDIO_API_KEY` -- Ключ API (если требуется)

**Случай использования:** Локальный инференс с OpenAI-совместимым API.

## Резервный провайдер

Система поддерживает автоматическое резервирование при сбое активного провайдера:

- `LLM_FALLBACK_PROVIDERS` -- Разделённый запятыми список резервных провайдеров
- `LLM_AUTO_FALLBACK` -- Включить автоматическое резервирование (по умолчанию: true)
- `LLM_FALLBACK_TIMEOUT` -- Таймаут перед попыткой использования резервного провайдера (по умолчанию: 10 секунд)
